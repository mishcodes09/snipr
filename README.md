# 🧠 Snippet Manager

A simple and intuitive snippet manager that helps you store, organize, and quickly access your most-used code snippets.


## ✨ Overview

Snippet Manager is built for developers who want speed, structure, and flexibility when working with code snippets. 
With AI-powered translation, you can convert snippets across frameworks and languages in seconds (for example Vue to React, JavaScript to TypeScript, or backend logic to frontend equivalents).

---

## 🚀 Features

- Save and organize code snippets by category or tags
- Fast search across all snippets
- One-click copy to clipboard
- AI-powered code translation between frameworks and languages
- Lightweight and fast performance
- Clean, modern UI
- Suberbase Storage

---

## 🧠 AI Feature (Core Highlight)

Translate code instantly across frameworks and languages.

Examples:
- Vue to React
- JavaScript to TypeScript
  
This allows you to reuse logic instead of rewriting everything manually.

  
## 🧰 Tech Stack

* Vue.js 
* JavaScript 
* Tailwind CSS 
* AI API 
* Supabase

## ⚙️ Installation
Clone the repository:
git clone [https://github.com/your-username/snippet-manager.git](https://github.com/your-username/snippet-manager.git)
Navigate into the project:
cdd snippet-manager  
Install dependencies:
npm install  
Run the app:
npm run dev  
---
## 🛠️ Usage  
1. Create a new snippet  
2. Add a title, code, and tags  
3. Save it  
4. Search or filter snippets  
5. Use AI to translate code across frameworks  
6. Copy and reuse instantly
